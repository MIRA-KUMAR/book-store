# BOOKSTORE PROJECT

## Backend 

1. Initialise node 
    
    `npm init -y`

2. Install `express` and `nodemon`

    `npm i express nodemon`

3. Install `mongoose` library

    `npm i mongoose`


### Creating Database

1. Create a Database in MongoDB 
2. Connect to backend with username and password . Add a collection (book-collection)
